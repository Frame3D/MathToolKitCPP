Detailed Description

template<typename T>
class NVector< T >

A NVector<T> object represents the coordinates of a finite dimension vector x in an arbitrary base.

Date
03/05/2018
Author
samiBendou

Coordinates are stored in the form [x0, x1, ..., x(n-1)]. where [...] is a std::vector<T>, n is the dimension and (x0, x1, ..., x(n-1)) are the coordinates.

Featuring algebraical operations (E, +, *), swaps, shifts and classic vectors generator such as ones, zeros...

This object is the base object of every other object in NAlgebra module. It inherits from std::vector<T> so it's kind of memory support for every object in NAlgebra module.

We will often refer to the following variable in this page :

  • x/u : this vector.
  • s : a scalar (T)
