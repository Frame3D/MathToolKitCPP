Detailed Description

AESByte class is a representation of an element of Gallois's finite field \( GF(2^8) \).

Date
15/10/2018
Author
samiBendou

providing algebraical operation on the specific field. and interfacing with integers primitive types. For more details go to https://nvlpubs.nist.gov/nistpubs/fips/nist.fips.197.pdf.
