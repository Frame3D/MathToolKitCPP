◆ rows()

template<typename T >
vector< NVector< T > > NPMatrix< T >::rows ( ul_t  i1 = 0,
ul_t  i2 = MAX_SIZE 
) const

Create an array containing the rows of the matrix.

Parameters
i1 First row to be taken.
i2 Last row to be taken \( i_1 \lt i_2 \).

The behavior of rows(,) is the following :

  • rows() returns all the rows.
  • rows(i1) returns the rows \( [R_{i1}, R_{(i1+1)},..., R_{(n-1)}] \)
  • rows(i1, i2) returns the rows \( [R_{i1}, R_{(i1+1)},..., R_{i2}] \)
Returns
Return an array containing the rows of the matrix as Nvector.
